# 🩺 DiagnoseAI – AI Powered Medical Assistant

DiagnoseAI is a **Generative AI powered healthcare assistant** that helps users understand medical information, analyze reports, and assess potential health risks using AI.

The system combines **Machine Learning, Retrieval-Augmented Generation (RAG), and Large Language Models (LLMs)** to provide intelligent healthcare assistance.

⚠️ **Disclaimer:** This project is for educational and research purposes only and does not replace professional medical advice.

---

# 🚀 Features

## 🤖 Multilingual Medical Chatbot
- Ask health-related questions
- Supports multiple languages
- Powered by **Google Gemini**

## 📄 Medical Report Summarization
- Upload medical reports
- AI summarizes complex reports into simple language

## 📚 Chat with Medical PDFs (RAG)
- Upload medical reports in PDF format
- Ask questions directly from the report
- Uses **FAISS vector database + Gemini embeddings**

## 🩸 Disease Risk Prediction
Machine learning models predict potential health risks:

- Diabetes Risk Prediction
- Lung Cancer Risk Prediction

## 📊 RAG Evaluation
Evaluate the RAG pipeline using **RAGAS metrics**:

- Context Precision
- Context Recall
- Faithfulness
- Answer Relevancy
